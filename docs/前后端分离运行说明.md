# 前后端分离运行说明

## 1. 端口约定

1. 前端：`5173`
2. 后端：`8080`

## 2. 启动后端

在项目根目录执行：

```powershell
cd "E:\山东科技大学毕业设计\backend-spring"
$env:DB_PASSWORD="你的MySQL密码"
mvn spring-boot:run
```

访问：

1. [http://localhost:8080/api/v1/health](http://localhost:8080/api/v1/health)
2. [http://localhost:8080/api/v1/meta/modules](http://localhost:8080/api/v1/meta/modules)

## 3. 启动前端

另开一个终端，在项目根目录执行：

```powershell
npm.cmd --prefix frontend run dev
```

访问：

1. [http://localhost:5173](http://localhost:5173)

## 4. 当前联调说明

1. 前端接口基地址定义在 [config.ts](E:/山东科技大学毕业设计/frontend/src/config.ts)
2. 默认请求 `http://localhost:8080/api/v1`
3. 后端已开启跨域，支持本地前后端联调
4. 前端使用 Vue 3 + Vite + Element Plus + TypeScript 构建

## 5. 当前可演示内容

1. 系统概览页
2. 用户管理示例数据
3. 周反馈任务示例数据
4. 周反馈与实时反馈列表
5. 新增实时反馈表单

## 6. 生产构建

```powershell
npm.cmd --prefix frontend run build
```
